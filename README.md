# van-build — CAD-каркас салона RAM ProMaster 1500 136" High Roof

Параметрическая **каркасная** модель салона кемпера на базе RAM ProMaster 1500
(колёсная база 136", высокая крыша; кузовной близнец — Fiat Ducato L2H2), построенная
во FreeCAD по 3D-скану реального кузова.

## Подход

- **CAD = только каркас** (к нему крепится оборудование). Обшивку в CAD не моделируем.
- **Обшивка = сам 3D-скан** кузова (меш), полупрозрачный, лежит поверх каркаса.
- Внутренний объём — не «коробка», а **сложная форма, снятая со скана**: гнутые борта,
  скруглённое плечо, округлая крыша. Рёбра каркаса — гнутые обручи по контуру скана.

Так совмещаются визуальное сходство (скан) и проектопригодная CAD-геометрия (каркас).

## Система координат

Ноль — пол у порога **задних** дверей, на центральной оси.
`X` = длина (0 сзади, +X к кабине); `Y` = ширина (0 центр, +Y пассажир); `Z` = высота (0 пол).

## Файлы

| Файл | Что это |
|------|---------|
| `van_v0.py` | Скрипт-генератор модели (правишь `PARAMS` → пересобираешь) |
| `van_v0.FCStd` | Собранный документ FreeCAD (каркас + меш-обшивка) |
| `Ducato_L2H2_body_reference.stl` | Чистая наружная оболочка кузова (нужна скрипту) |
| `Ducato+L2H2+lit+superpose2.stl` | Плотный скан целиком (с мебелью/человеком) |
| `Ducato+L2H2+lit+superpose2.skp` | Исходная SketchUp-модель |
| `PROJECT_MEMORY.md` | Сводная память по всему проекту (электрика, кузов, 3D и т.д.) |

## Сборка

Headless:

```bash
~/Applications/FreeCAD.AppImage --console ~/van-build/van_v0.py
```

Скрипт измеряет профиль борта из `Ducato_L2H2_body_reference.stl`, строит каркас,
кладёт скан как обшивку и сохраняет `van_v0.FCStd`.

## Статус

Габариты оболочки и форма свода **измерены по скану**. Точные позиции рёбер, проёмов
дверей и колёсных арок — пока SEED-заглушки; заменяются на реальные значения после
ручного промера фургона (см. раздел про промерку в `PROJECT_MEMORY.md`).
